# ⚖️ SKR Legal Platform

A modern full-stack legal services web application built with **React** and **Firebase**, designed to provide a professional online presence for legal awareness, blogging, and client engagement.

🌐 Live Website: [https://skrlegal.in/](https://skrlegal.in/)

---

## 📌 Overview

SKR Legal Platform is a production-ready web application that enables:

* Legal service presentation
* Blog publishing and management (CMS-based)
* Secure authentication for admin access
* Responsive UI for all devices

It is designed to improve legal awareness and provide structured access to legal content in an easy-to-use interface.

---

## 🚀 Features

### 🔐 Authentication System

## ⚙️ Setup Instructions (Local Development)

### 1️⃣ Clone Repository

```bash
git clone https://github.com/your-username/skr-legal-platform.git
cd skr-legal-platform
```

### 2️⃣ Install Dependencies

```bash
npm install
```

### 3️⃣ Setup Firebase

Create a `.env` file and add your Firebase config:

```
REACT_APP_API_KEY=your_api_key
REACT_APP_AUTH_DOMAIN=your_auth_domain
REACT_APP_PROJECT_ID=your_project_id
REACT_APP_STORAGE_BUCKET=your_storage_bucket
REACT_APP_MESSAGING_SENDER_ID=your_sender_id
REACT_APP_APP_ID=your_app_id
```

### 4️⃣ Run Development Server

```bash
npm run dev
```
